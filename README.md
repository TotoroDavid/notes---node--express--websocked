# Socket.io CRUD

This is a simple CRUD web application in real-time using Socket.io and Nodejs

### Installation

```
cd websockets-nodejs-crud
```

```
npm install
```

To execute in development mode

```
npm run dev
```

to build the project for production

```
npm run build
npm start
```
